# IPL Data Analytics Project

## Project Overview

This project analyzes Indian Premier League (IPL) cricket data using SQL and Tableau to generate business insights related to teams, players, venues, toss decisions, and season trends.

The project focuses on data cleaning, SQL analysis, KPI generation, and interactive dashboard creation.

# Tools & Technologies Used

* Tableau
* Excel

# Dataset Used

* matches.csv
* deliveries.csv

# Project Objectives

* Analyze team performance
* Analyze player statistics
* Study venue trends
* Analyze toss impact
* Generate season insights
* Create interactive dashboards

## 3. Venue & Match Analysis Dashboard

# Key Insights

* Mumbai Indians were the most successful IPL team.
* SK Raina scored the highest runs.
* SL Malinga took the highest wickets.
* Toss decisions had a noticeable impact on match results.
* Wankhede Stadium hosted the highest number of matches.
